Тестовое задание Описание функционала который требуется реализовать в ходе решения текущей задачи

Необходимо разработать веб-приложение с использованием Angular из двух страниц
(каждая страница должна открываться по-собственному url):

Списка стран

    Должен быть реализован следующий функционал:
  • Получения списка стран по API (/v1/geo/countries) серверной пагинацией
  • Отображение полученной информации в таблице
  • Фильтрация (поиск) по наименованию
  • По нажатию на иконку перехода (в ячейке ID страны) открывать страницу городов с предустановленным фильтром по стране

Списка городов

  • Должен быть реализован следующий функционал:
  • Получения списка стран по API (/v1/geo/cities) серверной пагинацией
  • Отображение полученной информации в таблице
  • Фильтрация по стране (select/dropdown) и наименованию (поиск)
  • По клику на иконку просмотра - открывать в popup для отображения информации по городу
  • По клику на иконку редактирования - открывать в popup для редактирования информации по городу

Бонусом будет:

    Валидация (для полей население, дата основания, долгота, широта, добавить валидацию на допустимые значения) и отображение ошибок на полях формы редактирования
  • Сортировка таблиц (Страны - Наименование, Код страны) (Города - Страна, Наименование, Регион, Население)
  • Выбор количества элементов в таблице
  • Отображение отредактированной информации (в таблице и popup-ах) в приоритете с полученной по API
  • Локализация интерфейса (русский, английский)
